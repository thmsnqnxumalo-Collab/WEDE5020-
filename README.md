A picture slideshow was added on the home page, just under the welcome section. It shows photos of happy clients in suits and dresses, plus shoes — to show the kind of work Symons Tailoring does.

*3. Search bar added on the Services page*  
A search box was added at the top of the Services page. The visitor can type a word like "dress", "shoe" or "curtain" and the page will only show the services that match what they typed. A message under the box tells them how many services matched.  

*4. JavaScript — one shared file*  
There is only one JavaScript file (JS/main.js) and every HTML page links to it.  
New code was added to that same file for:  
● Running the slideshow (auto-play, buttons, dots)  
● Running the live search on the Services page  
The older features are still there: highlighting the current menu link, updating the year in the footer, smooth scrolling, form checks, and stopping past dates being picked, abstracted previous JavaScript code from other pages and added it to the external JavaScript folder/file.  

*5. Stylesheet updates*  
New CSS was added at the bottom of CSS/Stylesheet.css for the slideshow (layout, captions, arrows, dots) and for the search bar (input box, focus colour, helper text). The rest of the design was not changed.
